发布于 2020-11-05 | 分类于 Thoughts 🍉 | 6分钟 | 1636字数 | 浏览量：

认识我自己

哲学史的意义，是让我们不再迷信，树立问题意识和批判意识

哲学的基本认识

哲学的发端

远古人类使用神话解释自然界，当人们不满足用命运解释现象时，人们开始寻求必然性和规律性，开始科学的思维。

  • 哲学不是***科学***，更不是***一个科学***
  • 哲学是理论化的世界观
  • 哲学需要使用有限的，日常语言，赋予自己的含义，描述无限的理想世界
  • 哲学要讲理性，要调和理性和信仰

终极关怀

终极关怀需要解决人在有限生命当中精神寄托的问题

张岱年先生指出，古今中外的终极关怀有三种类型：1、皈依造物主的终极关怀；2、返归本原的终极关怀；3、发扬人生之道的终极关怀。

终极关怀 终极关怀，源于人之存在的有限性，源于人之企盼生命存在的无限性，它是人类超越有限、追求无限以达到永恒的一种精神渴望。对生命本源和死亡价值的探索构成人生的终极性思考。这是万物灵长——人类独具的哲学智慧，寻求人类精神生活的最高寄托以化解生存和死亡尖锐对立的紧张状态，这是人类超越性的生死价值追求。

什么是哲学

哲学总是要首先建立一个第一原理 / 最高原则，并以此推演整个世界

哲学的开始

  • 从终极关怀开始：超越有限、追求无限以达到永恒
  • 对人生所能通达的最高理想的追求

哲学的特点

  1. 哲学不是一个哲学 / 哲学不是科学
  2. 哲学是理论化的世界观

西方一个普遍的认识

知识具有普遍必然性

而感觉和经验是相对的，偶然的

先验论/ 经验论的争论

哲学的作用

哲学要讲理性，要调和理性和信仰

哲学拥有理论，但关注终极关怀的问题

哲学需要使用有限的，日常语言的基础上，赋予自己的含义，描述无限的理想世界

“我思故我在”

  • 在知识真假难辨的情况下，肯定的判断都无法确证自身的合理性
  • 笛卡尔采用怀疑方法，确定了近代哲学理性的地位

Being v.s. I

笛卡尔把 我思故我在当作研究存在的形而上学的第一原理，从而完成了主体性的觉醒

主体和客体的区别，让 认识论的深入研究得以进行

形式上的主体：我

如何批判

小说中在思想的人物，他们是主体吗？捏造出来的thinking，也有独立的客观实体支撑吗？

不要在示例过少的时候进行总结

那样叫过拟合

如何看待哲学家

  • “可以怀疑哲学家的理论，但是基本不需要怀疑哲学家的智商”

如何看待唯心主义和唯物主义

唯心主义哲学是错的么？ - 难得糊涂的文章 - 知乎
https://zhuanlan.zhihu.com/p/130279854

武断地说“唯心主义错误”并不是好的讨论方式，反而显得很荒谬。哲学无定论，它给予我们的是一种思辨精神和思维体系，而非简单的结论。同时，哲学需要在相互辩论、互相诘难中澄清各自观点。

关心一个哲学家是唯心还是唯物的不是不重要，我们更关注的是他们的思想对我们什么启发。（否则）就是带有极左的倾向。

张志伟老师的观点

西方哲学史当中实际上并不存在唯我论 / 意识先于物质而第一 / 主观唯心主义的思想，这是很违背常识的

唯物 / 唯心的争论，不是一个层面上的争论，而是在

  • 时间性
  • 逻辑意义先后

如何看待不可知论

其实，人类已经完全理解了神经生物学，并且制造出了计算能力足够强的量子计算机。有一群科学家已经能够通过微模拟神经电路将一个人的大脑连接在超级计算机上面，当然，科学家也需要一些实验者来进行这样的实验。你或许以为我现在想问你愿不愿意参加到这个实验中来，但是事实上，我只是来告诉你，你已经参加了这个实验，你现在就是一个缸中之脑，你的记忆被我们改写了，所以你不记得自己参加过了这样的实验。你又要如何否认这一点呢？

如何理解“先天判断”

如何反驳不可知论？ - 罗心澄的回答 - 知乎 https://www.zhihu.com/question/20271625/answer/14566346

当我们证明了高维空间的存在，我们也无法感知。更何况，我们是以我们被赋予(Programmed)的认识形式去推理和演绎的。
置信度可以无限接近1，但不会是1。同时，因为我们要认识世界和改造世界

何为“知”
突然想到一个概念，就比如我坐在图书馆，说实话，我不会在意面前的茶杯里装的水的物自体是什么，但是我知道我想喝水，我的面前有水
可感的水解可感的我的渴

那个无法感知的世界，对我来讲没有很大意义
我所追求的理想和自由，不是超验的、只存在于幻象而不可感的自由，而是经验的，可以通过认识和改造世界可感的自由。

哲学就是理论化的世界观，我站在自己的立场上看世界，我关注可感世界。

你困惑吗？

并且自由 = 选择 = 承担后果

自由

    • 人的自由是无条件的
    • 筹划和选择
  • 但是自由的实现是有条件的

“剥夺自由”

自由无法剥夺
