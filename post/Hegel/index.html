古典哲学的终结

哈哈黑格尔中学喜欢看市民小说​网文😂

叔本华：我在读荷马的时候，黑格尔在读什么哈哈哈哈哈哈

hhh

人类自身像这样地被尊重就是时代的最好标志，它证明压迫者和人间上帝们头上的灵光消失了

——黑格尔

  • 《精神现象学》
  • 《逻辑学》
  • 《哲学科学全书纲要》
  • 《法哲学原理》

谁要想理解黑格尔，只能靠他自己

有多少个研究黑格尔的人，就有多少个黑格尔

  • 包罗万象、内容丰富
  • 辩证法、留下想象空间
  • 从发展的角度看事物
    • 黑格尔尽量把自己引出去，让事物自身运动展现

认识是动态的

理性批判

黑格尔：批判本身是否客观；康德是否在用理性批判理性；如此批判，根据何在；

  • 真正的批判应该是理性的自我批判

Knowledge=f(Object)Object=g(Knowledge)Knowledge = f(Object)\\ Object = g(Knowledge) Knowledge=f(Object)Object=g(Knowledge)

  • 在认识当中，知识和对象相互影响，矛盾运动

  • 认识是一个过程

康德是对理性认识进行了静态的结构分析，但是黑格尔将认识看作发展的过程

  • 运动是绝对的

👏我大呼过瘾！

比如说一朵花

  • 既指代当下的这朵花
  • 又指代其他时刻的这朵花

考察思维形式已经是一种认识历程了，所以，我们必须在认识的过程中将思维形式的活动对于思维形式的批判结合在一起。我们必须对于思维形式的本质及其整个的发展加以考察。思维形式既是研究的对象，同时又是对象自身的活动。因此可以说，这乃是思维形式考察思维形式自身，故必须由其自身去规定其自身的限度，并揭示其自身的缺陷。这种思想活动便叫做思想的"矛盾发展"（辩证法）

统一认识论和本体论

过往的哲学家们，很少把个人置身于世界当中

黑格尔站在了更高的层面

人就是宇宙的一部分，人就是宇宙的产物

🙌我大呼过瘾！

不要把我们和这个世界分离开看待，我们就在这个世界之中

  • “我们不认识这个世界”
    • 世界和我发生了矛盾
    • 这本身，就是世界发展变化的一部分
  • 如果发生了内讧
    • 球员之间发生了矛盾
    • “这个球队不团结”

个体关乎整体

我认识了世界，世界也会变得更加和谐

认识论不单纯是认识论
人类是自然进化的最高阶段，那么人类对世界的认识，也是世界的自我认识

认识世界的过程，既是人类精神的伊利亚特，也是人类精神的奥德赛

既是探险，也是还乡

我们既在认识世界，也在认识自己

  • 在黑格尔看来，人类精神就是宇宙绝对精神的具体体现
  • 人类的认识活动，相当于替宇宙进行自我认识
    注意黑格尔的唯心主义思想

❓绝对精神为什么要认识自身

实体即主体

斯宾诺莎：实体即自因

万物都不是僵死的，而是发展变化，并处在发展变化的世界之中的

康德：主体能动

不仅仅人在开展自己的运动，万物也在同等地运动

自我实现

黑格尔：实体作为主体的能动性就表现在它自身之中包含着纯粹的否定性

矛盾—否定性质的、对立面的统一—运动发展的动力

***实体的运动就是以终点为目的，自己展开自己，自己完成自己的“圆圈”***

潜在的东西，自我剖分产生对立面，展开为自然，达到人类精神，重新认识了自然。

否定之否定：从更高层面回到了自身。自然就是自己的原因

是故《易》有太极，是生两仪。两仪生四象。四象生八卦。

《易传·系辞传上》

没有实现自身之前的现实是潜在的，实现自己之后才能叫做真正的现实

辩证法

从苏格拉底开始的概念抽象，让我们一直生活在抽象的观念之中，与自然隔着观念的帷幕。很难体验和再现事物活生生的生命。

事物本身是由对立统一的各种因素组合而成，而近代形式逻辑、理性的发扬，让人们很难接受蕴含着矛盾冲突的实际事物

辩证的否定——扬弃/是与非的关系

  • 否定和终止
    • 联系的环节
  • 继承和发展
    • 发展的环节（列宁）

黑格尔认为“精神”是不变的

  • 我们看到一个杯子，杯子作为对象存在于我们的意识之中了

  • 我把我变成了对象，但我仍然保持自身的存在

  • “对象化自身，仍然保持自身同一性”

  • 扬弃，进而把握事物的本质

  • “前进回溯式的发展”

    越发展，本质展开的就越充分，越接近自身

黑格尔的地位

形而上学的集大成者

耗尽了古典哲学最后的生命力

  • 希腊人哲学思考的前提：宇宙是有机的整体，宇宙是有规律的，宇宙是可以认识的……

    这些在形而上学的发展过程中相继受到了质疑

  • 黑格尔

    宇宙是合理性的存在

  • 现代

    人类理性仅仅是人类看待世界的方式，世界本身是盲目的状态

釜底抽薪

古典哲学——宇宙和谐的完满性——的思维核心发生动摇

黑格尔辩证、发展的思想已经融入后世哲学，甚至人们心中。

这并不意味着人类不再关心形而上学的问题，而是探讨问题的方式和古典哲学不不再一样
